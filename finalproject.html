Final Project

This is my final project page.
The game Barrel Dodger was done by Abel Frias and Ethan Laramee.
As a user you have to dodge all the barrels falling from the skies. If you want to survive more time, you need to grab the extra lives falling, and to keep your score grab as much coins as possible.
Move using the arrow keys and don't get hit by a barrel! Have fun.
